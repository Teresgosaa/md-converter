# Конвертер документов в Markdown

Веб-приложение для конвертации PDF, DOCX, PPTX, XLSX, CSV и JSON файлов в формат Markdown.

---

## Требования

- Windows 10 / 11
- Python 3.10 или выше → https://www.python.org/downloads/
  - При установке обязательно поставьте галочку **"Add Python to PATH"**

---

## Установка

### Шаг 1. Скачать проект

```
git clone https://github.com/Teresgosaa/md-converter.git
cd md-converter
```

Или скачайте ZIP-архив с GitHub (кнопка **Code → Download ZIP**), распакуйте в удобное место.

---

### Шаг 2. Создать виртуальное окружение

Откройте терминал (командную строку) в папке проекта и выполните:

```
python -m venv .venv
.venv\Scripts\activate
```

---

### Шаг 3. Установить зависимости

```
pip install -r requirements.txt
```

Установка займёт несколько минут — скачивается Docling и его зависимости (около 1–2 ГБ).

---

### Шаг 4. Установить Tesseract OCR (только для режима сканов)

Tesseract нужен только если вы планируете конвертировать **сканированные PDF** (без выделяемого текста).

1. Скачайте установщик: https://github.com/UB-Mannheim/tesseract/wiki  
   Выберите версию **tesseract-ocr-w64-setup-5.x.x.exe**
2. При установке добавьте языковые пакеты: **Russian** и **English**
3. Установите в папку по умолчанию: `C:\Program Files\Tesseract-OCR\`

---

### Шаг 5. Установить Poppler (только для режима сканов)

Poppler нужен для работы с PDF в режиме OCR.

1. Скачайте: https://github.com/oschwartz10612/poppler-windows/releases  
   Выберите последний файл **Release-xx.xx.x-0.zip**
2. Распакуйте в папку `C:\poppler\`  
   Итоговый путь должен выглядеть так: `C:\poppler\Library\bin\`

---

## Запуск

```
.venv\Scripts\activate
streamlit run app.py
```

Браузер откроется автоматически по адресу `http://localhost:8501`

---

## Использование

1. **Загрузите файл** — PDF, DOCX, PPTX, XLSX, CSV или JSON
2. **Нажмите «Далее»**
3. На шаге «Конвертация»:
   - Для обычных PDF и документов — нажмите **«Конвертировать»**
   - Для сканированных PDF — включите флажок **«Сканированный PDF (OCR)»**, выберите движок и язык
4. **Скачайте готовый .md файл**

---

## Частые вопросы

**Приложение не запускается — ошибка "streamlit не найден"**  
Убедитесь что виртуальное окружение активировано (в строке терминала должно быть `.venv`).

**PDF конвертируется, но текст пустой**  
Это сканированный PDF. Включите флажок «Сканированный PDF (OCR)» на шаге конвертации.

**Ошибка при OCR — "tesseract не найден"**  
Проверьте что Tesseract установлен в `C:\Program Files\Tesseract-OCR\tesseract.exe`.
